A Quick-Start guide to using the INTREPID Server for inference of evolutionarily conserved residues

The INTREPID server has been designed to be easy to use, providing a familiar sequence-based interface to complex phylogenomic analysis.

To begin an analysis, paste a FASTA-formatted protein sequence in the sequence box (Fig. 1). Optionally, you may provide your email address, in which case an email will be sent to you when your results are available.

Click the Submit button to start the analysis.

Figure 1: Paste a protein sequence in the sequence box and click Submit to start an analysis.

When INTREPID starts, a URL link will be provided (Fig. 2). You can click on this link at any time to check the progress of your analysis. You can also bookmark this link to check at a later time.

As INTREPID runs, information about the progress of your analysis is printed to the screen, which will be refreshed every 15 seconds (Fig. 2).

Figure 2. The INTREPID server provides a bookmark link to check progress at a later time. The progress of the analysis is displayed as a running log.

When INTREPID has finished, a link to your results will be provided (Fig. 3). If you supplied your email address, this link will also be emailed to you.

Figure 3. A results link is provided when INTREPID has completed. The output log is available for inspection.

Following this link will display the results of your analysis (Fig. 4), which include a link to the raw INTREPID results as a text file, a plot of conserved residues on a 3D structure of your query sequence (if a 3D structure was available or could be constructed using comparative modelling), and a plot of the INTREPID conservation score for each position in the query sequence.

Figure 4. The INTREPID results viewer coordinates the display of significantly conserved residues on the 3D protein structure and across the query sequence.

The INTREPID results viewer uses a conservation score cutoff to highlight significantly conserved residues in both the structure and sequence displays. Residues with a conservation score at least as large as the cutoff are displayed in blue.

The conservation score cutoff can be changed by moving the slider below the 3D structure (Fig. 5). (If no 3D structure is available, this slider is to the left of the sequence display.) Dragging the slider changes the score cutoff; releasing the slider refreshes the displays with the new cutoff value.

Figure 5. Manipulating the slider highlights more or fewer conserved residues as significant (colored blue).

Thank you for taking the time to explore the INTREPID evolutionary conservation inference server. We hope you find the server both useful and easy to use.
